### 1. [Ed-Fi API Migration](case-studies/edfi-migration.html)
**Platform Strategy · EdTech SaaS**

**Frameworks used:** Options analysis, TCO modelling, stakeholder alignment, phased rollout  
**Outcome:** 42% onboarding time reduction · TEA compliance ahead of deadline · CSAT 72→83

### 2. [PRD: Unified Cloud Pack Conversion](case-studies/prd-cloud-pack.html)
**PRD · SaaS Commerce · VMware**

A working Product Requirements Document for the Unified Cloud Pack conversion feature on VMware's subscription commerce platform. Covers problem statement, user personas, user stories (MoSCoW), acceptance criteria (Given/When/Then), success metrics, RACI, risks, and open questions.

**Scope:** 40,000+ eligible VMware customers · 4 cross-functional teams
